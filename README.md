#智能跟随车APP
---
本项目为智能互联比赛项目

##概述
###项目介绍
本项目想通过手机来控制小车的运动，来解放人们的双手，方便人们生活。
###项目功能
> 通过项目归纳总结，预计有如下几点功能
> 
1. 蓝牙搜索功能
2. 蓝牙连接功能
3. 蓝牙发送数据
4. 可保存搜索到，连接过的设备
5. 可对相关设备进行个性化操作
